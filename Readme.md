# Cat Lock
Ubuntu screen lock with a randomized cat carousel

![cat](docs/cat.jpg "cat")

#### Features
  Cat img downloading
  Cat img carousel
  Locking all keys
  Password unlock
  Emergency unlock

### Instructions
First initialize a .env file with your values, following the .env.example.

#### Install the project locally:
   ```bash
   pip install .
   ```
#### Download the cat photos:
   ```bash
   cat-lock download
   ```
#### Run the application from anywhere in your terminal:
   ```bash
   cat-lock
   ```
To exit the carousel simply write your chosen password

### Emergency
In case of an emergency lockdown press the Esc key 100 times.
